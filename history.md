# 開發歷程 (History)

## [0.1.1+3] - 2026-01-02
### 新增
- **存檔覆蓋確認機制**：
  - 在主畫面點擊「挑戰模式」或「休閒模式」按鈕時，若偵測到已有遊戲紀錄，系統會先彈出確認視窗，避免誤觸導致舊進度被覆蓋。
- **畫面展示區塊**：
  - 在 `README.md` 中加入了視覺風格展示區塊，預留三種模式（新中式、科技暗色、卡通可愛）的展示位置。

### 優化
- **版本管理**：
  - 更新 `pubspec.yaml` 版本號至 `0.1.1+3`。
  - 確保 UI 與自動化建置系統同步識別新版本。

### 建置
- 成功發布 Release APK：`idiom_puzzle-v0.1.1-20260102-release.apk`。
